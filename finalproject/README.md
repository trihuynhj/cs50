## PICPRANKCS50
1. **Video Demo**: <https://youtu.be/QEzhvrqwjpQ>

2. **Description**:
    - Implementation: PicPrankCS50 is a simple web application built with Python (Flask framework) on the server side, HTML/CSS and some basic JavaScript on the client side. The program also utilizes the SerpAPI for Google Image Search engine, Python's SMTP library for sending email and some other minor libraries.
