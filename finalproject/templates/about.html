PicPrankCS50

PicPrankCS50 is a very rudimentary and tiny web application built with Python (more specifically Flask framework) on the server side, HTML/CSS and some basic JavaScript on the client side. For the image search engine I used SerpAPI (the free plan!), and I implemented the email sending function following this documentation. And of course, thank you to CS50 staff for their awesome IDE's built-in SQLite!
Admittedly, I utilized the MVC model and some initialization code from Problem Set 9 (Finance) to quickly get the program started, because I wanted to focus on the implementation of the core idea of the application. I'm not sure if it's against the rules, but I hope not haha (nervous laugh). And I hope you at least have some fun with it!
